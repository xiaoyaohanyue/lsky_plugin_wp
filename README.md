
## 环境依赖
* PHP >= 7.4
* fileinfo（必须）
* imagemagick (可选)

## API格式

* V1
域名/api/vi
例如：https://test.com/api/v1
* v2
域名/api/v2
例如：https://test.com/api/v2

## 使用方式

下载最新release的zip包，建议直接前往文件系统操作，解压到plugin目录下，确保本插件目录名为`lsky_plugin_wp`
